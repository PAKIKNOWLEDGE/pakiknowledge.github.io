Thinkpad T430 Hackintosh 以及其他心得

0 前言

首先很遗憾的通知大家，那台之前被我折腾过的松下 sz6 不幸的坏了。鉴于松下的配件难以替换还有在国内要找人修的难度实在太高，所以我遗弃了它） 接下来，鉴于我一向来不喜欢用游戏本办公和写作业的缘故，（包括这篇文章也在 t430 上写的）所以这意味着我必须物色一台新宠来满足我的需求。那么买全新肯定是不可能的了，实则就是，在电子硬件的选择与购买上，我觉得现在的情势突出两字：挣扎。（请注意我的用词和定语，我们就现在把它留下，因为后面会继续提到的。）抛开新闻稿上写的所谓什么“算力潮”和什么“供应链”云云，这些带来的实际客观情况就是，全新内存（DDR4/5) 和存储（包括固态 ssd、机械硬盘 hdd）都是全面四五倍的涨幅。但是在目前，DIY 装机对我来说是一种伪需求，但是存储的升级是一种迫切的需求。哎呀！这让我当时在隐星 p15 游戏本的内存选配犯下的错（指没有选择 1t ssd 而是 512g ssd）的代价更大了——因为我发现：

我换不起固态来装《剑星》还有其他的东西了。我操。

总的来说，我最后选择了 Thinkpad T430。明说吧，我一开始是想选择 X220/X230 的。但是阴差阳错之间我竟然没有这么做，但是总的来说我的确选择了 Thinkpad。这有三方面原因，一是 Thinkpad 并不像松下是小众猎奇之物，至少键盘坏了可以很方便的换，这也是基于 Thinkpad 庞大的用户基数的原因。再者，Thinkpad 大多数保留了方便的可拓展性设计。譬如为内存和硬盘位预留了方便显眼易拆卸的盖板和螺丝钉，（反观松下那破内存疑似板载）还有可拆卸电池（在这种特别老款的笔记本上，除非卖家有提前换过，否则大概率“附赠”一块千疮百孔/几近不可用的老爷电池）便于更换。最后，Thinkpad 的键盘手感和小红点让文字工作更加顺手。虽然 x220/t420 的键盘上永恒经典，在它两的后代 X240/T430 因为更换了新的键盘材质的原因常被评为拉完了，但是再这么说也比松下 sz6 那块反人类纯招笑日文键盘好。（更何况，它坏了。Thinkpad 以耐用著称，而松下疑似站在它的对立面）

1 Second choice

那么就来看看这台斥巨资（实则花费 370 元）买到的 Thinkpad T430。首先从成色上看，A 面还不错，有掉漆两处，不妨事，用两张管人贴纸应付了了事。B 面（也就是屏幕）有一小处明显可见坏点在最左侧，但是显然不影响使用，我并不是屏幕敏感用户，我的隐星 p15 屏幕被压伤了一处我至今没有什么感觉。然后看键盘和触控板，也不是很脏，是一个很好的水平。最后看 D 面，在内存和硬盘的主板螺丝处有一个防拆贴纸，据卖家所说这批 T430 是单位采购退役下来的，但是显然被翻修过。从配置上来看，这台 T430 搭载着 ivy bridge 架构的三代 i5、8g ddr3 内存、还有一块 300G 的….机械硬盘。从重量上看，裸机大概 2.0-2.3kg。开合难易度部分，t430 采用了锁扣设计，需要横推锁扣滑块才可以把盖子打开，不过呢，开锁之后可以很轻松的进行单手开盖。

需求

分析我纯文字办公的需求来看，虽然二代 i5 相较于之前搭载七代 i3 的松下 sz6 来说无异于是性能降级，但是在需求上看完全是足以应对，因为我买个本基本就是当打字机用的。内存管他是 ddr 几，只要是 8gb 基本是够用的。但是主要的问题在于硬盘，机械硬盘在如今确实会带来令人崩溃的体验，外加我有黑苹果的需求，所以一块固态硬盘就成刚需了。但是前面我说过，在如今硬件的选购时令人挣扎的，那么死马当活马医，我又斥巨资（实则为 60RMB) 购买了一条通电五百万小时也不知道是哪个犄角旮旯生产的二手 128g m.sata ssd。

加装固态硬盘

这台笔记本加装内存和固态还是比较简单的，所需的工具只有一支十字螺丝刀就可以了，当然，如果你有金属桌腿或者什么类似的金属的东西，那也可以利用起来，在你换零件之前摩擦两下，除除静电！我也是斥巨资（实则为¥3）在学校门口超市买到了妙妙工具，然后看到 D 面，有一个专门的，由四个螺丝固定的盖板，拧开就能自行加装硬盘和内存。（自带的大机械硬盘则不是在这个地方，这个地方是拓展存储用的）盖板的四个螺丝是防呆设计的，就算被拧到极限也不会掉下来，所以放心拧开，然后拿出固态硬盘，拧出 sata 插槽的固定螺丝（但是要注意，这颗固定螺丝可没有防呆设计，要是丢了就得哭天抢地了）看准金手指插槽斜四十五度怼进去，之后小心施力按平，再拧回插槽的固定螺丝就可以了。（注意到，这个金手指插槽显然也是防呆设计的，因为我在查阅和观看拆机视频的时候了解到，如果你翻错面了这个金手指是对不进去的。当然，所谓防呆不防傻，如果某些人用蛮力把插槽卡坏了那也犹未可知吧！ )

2 Hackintosh：饺子与醋

啥是醋

在压低了预算之后，我们购买到了这样老爷的硬件，正所谓有些东西越老越妖，硬件老不一定是坏事，虽然老硬盘通电时间长那肯定令人忍俊不禁，但是老硬件也就意味着在如今 macOS 全面转向自研 m 芯片的当下还有黑苹果的余地。算了，明说吧，饺子吃完了醋来了，我买老 Thinkpad 就是为了装黑苹果用的。在使用松下 sz6 的过程中我也是体验到了在纯文字工作的需求驱动下，macOS 提供更易用的界面和更加优秀的多任务处理能力，以及和苹果设备更好的联动 （虽然这个狗屁蓝牙驱动我至今也是没有修好用不了 airdrop） 大白话就是，macOS 不仅确实挺好用的还可以装逼。而且我在购买 t430 之前就已经考察过，网上已经有兼容性非常好、完成度非常高的 efi 可供使用。那么接下来就是安装的过程

安装流程与坑

黑果的引导一直以来都主要由 clover（俗称四叶草🍀）和 opencore（俗称 OC) 占主流。而 open core 有着更加频繁的更新、更加详尽的社区指南、更加先进的硬件检测和驱动自动装载程序，包括上次我在给 sz6 装黑果的时候也用的 oc，所以我这次依旧选择用 oc 来引导。（虽然随着苹果的封锁和弃用英特尔芯片的大趋势下，黑苹果差不多要走向生命的尽头了）

别呀，先等等！

注意到，efi 文件之所以叫 efi，在于 macOS 的引导实则是与纯 UEFI 引导强绑定的。而这台 t430 我拿到手的时候是用的传统启动（MBR) 且预装 Windows7 系统。win7 对 uefi 的兼容性说很差的，所以为了让 Windows 也兼容这个纯 uefi 的方案，我决定先把机械硬盘的 win7 格了换成 win10，然后再把 macOS 装到 ssd 上。但是考虑到，这机器确实上了年头，装原版的 windows10 确实有点压力，于是我上网找了一个精简版的系统，最后选择了这个叫Windows 10 Pro 22H2 19044/19045 二合一 轻度精简版 - 小修（2026.03.12）的系统，系统是 esd 格式的，在精简之下只有 1.5g 左右的大小。虽然说精简版系统可能有各种的问题，但是在这台本子上我肯定大部分时候都呆在 macOS 并且用合盖睡眠，所以 Windows 只需要等着不时之需就可以了。为了防止出问题，我还准备了一个 pe 盘，用的是微 pe。先用用 pe 里的分区助手转换为 gpt 分区表，创建 efi 分区，再用 winnt setup 把 esd 文件安装进硬盘，由于系统经过精简的原因，即使在机械硬盘上安装也非常快，开机之后发现系统啥都没有，连过渡动画也被精简了，但是目前没遇到什么功能问题，也非常流畅，这里也推荐有老硬件的朋友，不考虑 Linux 的情况下可以试试这些精简版镜像。

好的，回到正常黑苹果的安装。那么现如今黑苹果的安装其实已经非常简单。（在有适配的 efi 的前提下、即使没有，现在也能全自动生成）主要有三步：

  • 1 获得 efi 文件和镜像
  • 2 使用你的安装介质把 efi 装载，并引导进系统后安装
  • 3 后续配置、补全功能、开始使用

一般来讲，网上的教程大多数都推荐使用 u 盘作为安装介质。不过我觉得有些麻烦，于是我采用和上次在 sz6 上安装 macOS12 一样的方法，使用 rdr 镜像恢复安装。首先获取一份 rdr 镜像，然后把 efi 文件放入磁盘的 esp 分区中，再用 rdrive 软件把 rdr 文件恢复进硬盘后就可以了。这次我依旧选择了 macOS 12 Monterey 版本，因为本着好用就继续用的原则，虽然看见有些人在 t430 里安装了 macOS26，但是还是维持和 sz6 上的原状吧。

显卡驱动问题

这次安装的过程中，我首先尝试了来自 jozews321\T430-Hackintosh-Opencore 的这份 efi。这份 efi 本来的功能支持已经完全是完美的水平，但是由于 apple 在 macOS Big Sur 版本之后已经停止了对 HD 4000 集成显卡的支持，所以在引导系统之后需要用OC 老婆 其实是OpenCore Legacy Patcher(OCLP)进行打补丁之后才可以正确的调用显存并启用硬件加速。就在使用这份文件的时候，我虽然成功引导进入了系统，但是呢，不清楚是不是我 oclp 版本问题还是咋地，我的 patch 按钮竟然是灰色的。（当然，我事后才意识到可能是 oclp 的问题，因为 open core 这些插件拓展工具，包括什么 occ、oclp 都疑似和 open core 的版本一一对应，而这个问题在OpenCore Auxiliary Tools (OCAT )才被解决，ocat 没有 occ 那样和版本强绑定，这也导致我弃用 occ）这之后我又换了一份 ufi，是来自Xiaoheixu\Thinkpad-T430-s-OpenCorteBootloader，作者似乎是国人，不过联系方式均已失效。这份 efi 和上一份有两处不同，一是：作者将自己的网卡替换成了 Bcm94360HMB/Bcm943224HMP、二是，作者自己在 release 里附赠了一个 oclp。这解决了一个问题、又产生了另一个问题。这解决了 oclp 不可用的问题。果不其然，这次换用这个 efi 启动之后，成功用 oclp 驱动了 hd4000，启用了硬件加速，其他各项功能也没有太大的问题。

网卡驱动问题

但是，产生了新的问题，由于我没有和作者一样换用博通网卡，而是 Thinkpad t430 原生自带的Intel Centrino Wireless-N 2200（1x1 BGN），这导致其 efi 中的驱动和实际的硬件是不对应的，自然而然 Wi-Fi 和蓝牙也是瘫痪的状态。不过，作者在文档中也说，可以自行添加英特尔网卡和蓝牙驱动。此时，注意到，如果同时加载博通驱动和英特尔驱动，可能会出问题，所以我先在ocat中，删掉了有关博通Wi-Fi的东西，这之后，我置入了GitHub - OpenIntelWireless/itlwm驱动。（其实之前在松下用的是AirportItlwm，感觉不太稳定，换用了）不同于AirportItlwm调用macOS原生Wi-Fi管理，itlwm采用Heliport.app来管理Wi-Fi。虽然这会导致macOS菜单栏的原生Wi-Fi无法使用，但是，只需要手动启动程序，heliport会自动蹦上去菜单栏，达到和原生状态栏连接一样的效果。

实际体验与遗留问题

至此，在t430上安装macOS12的流程就至此结束了，这次的配置，比起在松下sz6上那次需要手动修复睡眠问题，t430的睡眠和唤醒倒是完美的工作。而触控板也是完美驱动上了（傻逼松下不知道哪个二逼卵子设计的圆盘触控板难用的要死还驱动不上真是这辈子有了）但是还是不太完美的，注意到，上面虽然修复了Wi-Fi驱动，但是没有修复蓝牙驱动呀！我尝试用相同的方法，把博通的驱动删了然后植入英特尔蓝牙驱动，结果系统炸了，吓得我再也不敢修了，又不是不能用.jpg 没了airdrop功能，至少我们还有微信文件传输助手呢。

在装好系统的实际体验上，还是有一些问题存在的。比如，itlwm驱动似乎跑不满我校园网的带宽，导致下载速度奇慢无比，原因不明，但是此问题需要控制变量，等我找另一个网再次尝试一下，然后才可得出结论。还有就是，这笔记本比起松下sz6恐怖的轻量化（800+-g），其2.2kg左右的重量外加上电源适配器，原先用手提电脑包就可以装上sz6和充电器，这次就必须上双肩包了。所以说，在便携性上，还是打了一些折扣，但是实际体验下来，也是完全不会有游戏本那么重的夸张。（

早知道买x220/t430s）

还有再说一下续航的问题。卖家声称电池还有91%健康度 我去，还有暗广啊 结果实测感觉根本不符合，而且出现了电量从10%跳到了2%的情况，所以我斥巨资（实则为75元）购买了一块全新的电池，在全程连接Wi-Fi开着vs code和Firefox-esr写这篇文章的时候，14:46开始，电量99%，中间不间断直到17:02剩下20%电量。理论续航不到3h，对于这老本来说还可以了，你不能指望ivy bridge和lunar lake竞争呀。

3 总结

买了三双了，质量非常好，与卖家描述一致，孩子们都特别喜欢吃，版型很好，面料舒适，尺码标准，不沾杯持妆久，橘色系更加显白，洗完也很柔顺头发也不油了，屏幕清晰运行流畅，敏敏肌用起来毫无压力，握感舒适不伤牙龈，每次炒菜都会放几滴，很满意的一次购物
