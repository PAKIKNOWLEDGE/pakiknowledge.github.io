LR2自动灯简易配置方法

排斥二次元的IR咱们就别用了！！111

Posted by     "pakiknowledge as lolikong" on Friday, January 6, 2023

前言：由于笔者在访问GitHub仍然遇到诸多困难，所以请确保在开始配置之前您可以顺利的访问GitHub，此外仍然强烈建议开启或者保留代理软件作为最后手段。

1 gsm是什么，以及为什么这篇教程会有V2

gsm就是自动灯，这里不再赘述。那么之前已经写过一篇有关的教程了，为什么会有第二版呢？首先是次要的，就是原先的配置流程相对较为麻烦，而现在有了更简易方法。（特别感谢BMS群群友quailty，是他发现了绕过IR使用插件的方法）之后是主要原因，就是原先这个插件需要配合Bokutachi IR进行使用，而该网站的规则中居然骂萝莉是乐色，我觉得这是不可接受了，因为首先笔者就是萝莉控，这种IR不如别用了！

2 怎样使用

首先去这里下载bokutachi IR的启动器，再去这里得到LR2GAS.dll把所有文件放在LR2的根目录。

最后请确保BokutachiHook.dll BokutachiLauncher.exe libcurl.dll zlib1.dll和LR2GAS.dll这些文件在你的LR2跟目录（也就是说，要和LRHbody.exe处于同个目录）

之后是比较关键的一步

在LR2游戏目录创建一个叫BokutachiAuth.json(请注意后缀！！)的文件，之后在里面填入：

{
   "apiKey":"xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
   "url":"xxxxxx"
}

只要apiKey恰好40字节，url不是空就可以过校验(也就是说url填lolisaiko都行，哈哈！)

再次感谢quailty，他伟大的贡献了这部分决定性的内容。

这之后，启动BokutachiLauncher.exe就可以体验自动灯了！

然后贴一个上一篇文章的避雷部分，懒得重写直接复制了：

直接点击BokutachiLauncher.exe启动会直接闪退，但是对这个程序使用LEGUI之后就正常了，推测是转区一类的问题，请各位自行避雷。

最后感谢BMS群提供帮助的各位，也感谢代码作者，请不要吝啬您的star。希望大家都能愉快的使用自动灯插件，祝大家生活愉快！
